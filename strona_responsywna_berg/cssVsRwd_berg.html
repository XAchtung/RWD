CSS a RWD

CSS (ang. Cascading Style Sheets) dzięki nim można zmienić wygląd całej witryny, zmieniając tylko jeden plik.

RWD (z ang. Responsive Web Design) jest to technika projektowania stron WWW, aby ich wygląd i układ dostosowywał się automatycznie do rozmiaru okna urządzenia, na którym są wyświetlane.
